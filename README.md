# سوق الأفكار

منصة عربية تجريبية لبيع وشراء الأفكار وباقات التنفيذ الجاهزة، مبنية باستخدام `Next.js` و`TypeScript` و`Tailwind CSS`، وتدعم `RTL` بالكامل في كل الصفحات والعناصر.

## لمحة سريعة

- اسم المنصة: `سوق الأفكار`
- الوصف: منصة لبيع وشراء الأفكار وباقات التنفيذ الجاهزة
- اللغة الأساسية: العربية
- الاتجاه: `RTL`
- البيانات: محلية وهمية فقط، بدون باك إند حقيقي
- الحالة: نسخة `MVP` غنية بالمحتوى والواجهات وقريبة بصريًا من الإطلاق

## أهم ما يحتويه المشروع

- صفحة رئيسية غنية بالأقسام والبطاقات والآراء والأسئلة الشائعة
- صفحة تصفح أفكار مع بحث وفلاتر وفرز
- صفحة تفاصيل فكرة ديناميكية مع تبويبات ومراجعات ومحتوى مشابه
- صفحة بيع فكرة بنموذج طويل مع تحقق ورسالة نجاح
- لوحة تحكم مستخدم
- لوحة تحكم إدارة
- صفحات ملفات البائعين
- صفحات تسجيل الدخول والتسجيل واستعادة كلمة المرور
- صفحات إضافية: من نحن، تواصل معنا، الأسعار والعمولات، الثقة والأمان، مركز المساعدة، الشروط، المفضلة، السلة، إتمام الشراء، الإشعارات، 404
- حركات تفاعلية: `scroll reveal`, `hover lift`, `count up`, `animated bars`, انتقالات للمودال والتبويبات، وشريط تقدم أثناء التمرير

## طريقة التشغيل

### 1. تثبيت الحزم

```powershell
npm.cmd install
```

### 2. تشغيل المشروع محليًا

```powershell
npm.cmd run dev
```

ثم افتح:

```text
http://localhost:3000
```

### 3. بناء نسخة إنتاجية

```powershell
npm.cmd run build
```

### 4. تشغيل نسخة الإنتاج

```powershell
npm.cmd run start
```

## بنية المشروع

```text
app/
  admin/                    لوحة الإدارة
  auth/                     صفحات الدخول والتسجيل والاستعادة
  cart/                     السلة
  checkout/                 إتمام الشراء
  contact/                  تواصل معنا
  dashboard/                لوحة المستخدم
  favorites/                المفضلة
  help/                     مركز المساعدة
  ideas/                    التصفح والتفاصيل الديناميكية
  notifications/            الإشعارات
  pricing/                  الأسعار والعمولات
  sell/                     نموذج بيع فكرة
  sellers/                  ملفات البائعين
  terms/                    الشروط والأحكام
  trust-safety/             الثقة والأمان
  about/                    من نحن
  globals.css               التنسيقات العامة والحركة
  layout.tsx                التخطيط العام للموقع
  page.tsx                  الصفحة الرئيسية

components/
  auth/                     أغلفة صفحات المصادقة
  forms/                    النماذج
  home/                     مكونات الصفحة الرئيسية
  ideas/                    بطاقات وتبويبات وفلاتر الأفكار
  layout/                   الهيدر والفوتر والتخطيطات
  motion/                   مكونات الحركة التفاعلية
  ui/                       عناصر واجهة قابلة لإعادة الاستخدام

data/
  categories.ts             التصنيفات
  dashboard.ts              بيانات اللوحات والجداول
  ideas.ts                  الأفكار الوهمية
  reviews.ts                المراجعات
  sellers.ts                البائعون
  site.ts                   بيانات عامة مثل FAQ وآراء المستخدمين

lib/
  data.ts                   مجمع البيانات ومساعدات الوصول
  utils.ts                  أدوات عامة مثل تنسيق العملة

types/
  index.ts                  أنواع TypeScript المشتركة
```

## الملفات الأهم

- `app/page.tsx`: الصفحة الرئيسية
- `app/ideas/page.tsx`: واجهة التصفح
- `app/ideas/[slug]/page.tsx`: صفحة تفاصيل الفكرة
- `app/dashboard/page.tsx`: لوحة المستخدم
- `app/admin/page.tsx`: لوحة الإدارة
- `components/ideas/idea-browser.tsx`: الفلاتر والفرز
- `components/ideas/idea-detail-tabs.tsx`: تبويبات التفاصيل
- `components/forms/sell-idea-form.tsx`: نموذج بيع فكرة
- `components/motion/reveal.tsx`: كشف العناصر أثناء التمرير
- `components/motion/scroll-progress.tsx`: شريط تقدم القراءة
- `data/ideas.ts`: قاعدة الأفكار الوهمية
